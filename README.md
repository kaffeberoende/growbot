# growbot

Plant watering system based on Raspberry Pi, PiFace2 and soil moisture sensors.

Hardware:
  * Raspberry Pi 3 model b
  * PiFace input/output card with two relays
  * 6 * digital soil moisture sensors
  * TOBD: two pumps

Software:
  * Python web server
  * Python io logic
  * REST API for mobile client
  * Client for Android
